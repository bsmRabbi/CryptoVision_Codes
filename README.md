CryptoVision Dataset Code Repository
CryptoVision is an open-source project for building a research-grade cryptocurrency news sentiment and market dataset via automated data collection and analysis.
This repository contains streamlined, well-documented Python scripts focused on four major pipeline steps:

Data scraping

Coin type detection

Candle (market) data fetching

Sentiment analysis

Overview
The CryptoVision pipeline is designed to help researchers:

Collect cryptocurrency-related news from trusted platforms

Automatically identify and classify coin types mentioned in articles

Fetch high-frequency OHLCV (candlestick) market data corresponding to news publication times

Annotate news articles with sentiment labels using transformer-based models

These tools support the rapid construction of enriched crypto market datasets for sentiment analysis, price forecasting, and trend prediction tasks.
